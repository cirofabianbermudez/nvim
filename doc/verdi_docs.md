# Verdi notes

Verdi shortcuts

## Code view

| Shortcut          | Description  | Manual                                       |
| ----------------- | ------------ | -------------------------------------------- |
| `Alt + Shift + D` | Go to Driver | Right click in a signal, **OneTrace/Driver** |
| `Alt + Shift + L` | Go to Load   | Right click in a signal, **OneTrace/Load**   |

## Code waveform view

| Shortcut | Description            | Manual                 |
| -------- | ---------------------- | ---------------------- |
| `G`      | Add signal to waveform | **Signal/Get Signals** |
